# Auto Adornos RD — CRM Dashboard Demo

Multi-channel customer relationship management dashboard for Auto Adornos RD.

## Features

- **Unified Inbox** — WhatsApp, Facebook, Instagram, TikTok messages in one place
- **Client Cards** — Name, phone, vehicle, request details
- **Status Tracking** — New / In Progress / Responded / Lost
- **Dashboard** — Stats by channel, messages/day, response rate

## Tech Stack

- React 18 + TypeScript
- Vite (fast builds)
- Tailwind CSS (styling)
- Axios (API calls)

## Development

```bash
npm install
npm run dev
```

Visit http://localhost:5175

## Build

```bash
npm run build
```
